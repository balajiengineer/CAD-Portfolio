# 📌 Project Overview

This project was completed as part of a Senior Mechanical Engineer technical assessment.

The objective was to redesign an industrial farfalle pasta forming process capable of manufacturing pasta at **1/100th of the original size** while introducing only minimal modifications to the existing production system.

The assessment also required analysing manufacturing challenges, proposing practical engineering solutions, and developing a CAD model for a food-grade mechanical assembly.

---

# 🎯 Objectives

- Miniaturize the farfalle production process
- Identify major manufacturing challenges
- Develop practical engineering solutions
- Design manufacturable components
- Create complete CAD assembly
- Estimate component mass
- Select suitable food-grade materials

---

# 🔍 Engineering Challenges

## Challenge 1

Miniature pasta handling and positioning.

### Proposed Solution

High-precision forming dies with improved alignment features.

---

## Challenge 2

Maintaining dimensional accuracy at micro scale.

### Proposed Solution

Precision guide pins, tighter tolerances and controlled forming force.

---

## Challenge 3

Reliable forming without damaging delicate pasta.

### Proposed Solution

Optimized punch geometry and controlled compression mechanism.

---

# ⚙ Mechanical Design
